Introducción a PHP

PHP (Hypertext Preprocessor) es uno de los lenguajes de programación más utilizados en el desarrollo web del lado del servidor. Permite crear páginas dinámicas, procesar formularios, gestionar bases de datos y generar contenido en tiempo real.

Aprender PHP es un primer paso fundamental para quienes desean adentrarse en el mundo del desarrollo web, ya que combina sencillez, flexibilidad y una gran comunidad de soporte.

En esta página encontrarás una introducción a las estructuras básicas de control en PHP, que son la base para tomar decisiones y repetir acciones dentro de un programa:

  • If: ejecutar código bajo una condición.
  • Switch: seleccionar entre múltiples opciones.
  • While: repetir acciones mientras se cumpla una condición.
  • For: repetir acciones un número definido de veces.
  • Foreach: recorrer arreglos y colecciones de datos.

Con estos conceptos podrás comenzar a escribir tus primeros programas en PHP y comprender cómo los sitios web toman decisiones y procesan información.

If

Las sentencias if en PHP sirven para ejecutar un bloque de código de forma condicional, es decir, solo si se cumple una determinada expresión o condición que evalúa a verdadero (true), y esto permite al programa tomar decisiones, definir comportamientos distintos para diferentes escenarios y controlar el flujo de ejecución de la aplicación web.
Por Ejemplo:

<?php
$numero = 10;
if ($numero > 5) {
  echo "El número es mayor que 5";
}
?>
Explicación: En este ejemplo la condición $numero > 5 se evalúa como verdadera porque $numero vale 10. Por lo tanto, se ejecuta el bloque de código dentro del if y se muestra en pantalla el texto "El número es mayor que 5".

Switch

La instrucción switch sirve como una alternativa más limpia y organizada a múltiples declaraciones if-elseif-else para ejecutar diferentes bloques de código basándose en el valor de una única variable o expresión, esto especialmente útil cuando se necesita comparar la misma variable con muchos valores posibles y ejecutar una acción distinta para cada uno haciendo el código más legible y fácil de mantener.
Por ejemplo: 

<?php
$dia = "lunes";
switch ($dia) {
  case "lunes":
    echo "Primer dia de semana";
    break;
  case "martes":
    echo "Segundo dia de semana";
    break;
  case "Miercoles":
    echo "Tercer dia de semana";
    break;
  case "Jueves":
    echo "Cuarto dia de semana";
    break;
  case "viernes":
    echo "Quinto dia de semana";
    break;
  case "sabado":
    echo "Sexto dia de semana";
    break;
  case "domingo":
    echo "Septimo y ultimo dia de semana";
    break;
  default:
    echo "Día no válido";
}
?>
Explicación: La variable $dia contiene la cadena "lunes". El switch evalúa los casos y encuentra coincidencia con el primer caso ("lunes"), por lo que se ejecuta el código correspondiente y se imprime "Primer dia de semana". Si $dia tuviera otro valor que no fuera un dia correcto, se ejecutaría el default.

While

La instrucción while sirve para ejecutar repetidamente un bloque de código mientras una condición especificada sea verdadera, la condición se evalúa al principio de cada iteración, si es verdadera se ejecuta el código dentro de las llaves, si es falsa el bucle termina, esto es útil cuando no se sabe exactamente cuántas veces se repetirá el código pero sí se sabe bajo qué condiciones debe continuar o detenerse. 

<?php
$i = 1;
while ($i <= 100) {
  echo $i;
  $i++;
}
?>
Explicación: El bucle comienza con $i = 1. Mientras la condición $i <= 100 sea verdadera, se ejecuta el bloque, mostrando el valor de $i y luego incrementándolo con $i++. El ciclo imprime los números del 1 al 100 en pantalla.

For

La instrucción for en PHP sirve para ejecutar un bloque de código un número específico de veces siendo ideal cuando se conoce de antemano la cantidad de iteraciones, esto se compone de tres partes: la inicialización (que se ejecuta una vez al inicio), la condición (que se evalúa antes de cada iteración) y la expresión de actualización que se ejecuta al final de cada iteración), esto útil para recorrer estructuras de datos como vectores y matrices o simplemente para realizar una tarea repetidamente. 

<?php
for ($i = 1; $i <= 10; $i++) {
  echo $i;
}
?>
Explicación: Este bucle for inicia con $i = 1, y se repite mientras la condición $i <= 10 sea verdadera. En cada iteración, $i aumenta en 1 gracias a $i++. Como resultado, se imprimen en pantalla los números del 1 al 10.

Foreach

La estructura foreach sirve para recorrer cada elemento de un array u objeto Traversable, ejecutando un bloque de código para cada elemento, su uso es ideal para interactuar con colecciones de datos ya que permite acceder de forma sencilla al valor (y opcionalmente a la clave) de cada elemento en cada iteración, haciendo el código más legible y limpio que los bucles for o while tradicionales. 

<?php
$colores = ["rojo", "verde", "azul"];
foreach ($colores as $color) {
  echo $color;
}
?>
Explicación: El foreach recorre el arreglo $colores, tomando cada valor en la variable $color en cada iteración. En este ejemplo, se mostrarán en pantalla las palabras "rojo", "verde" y "azul" de manera ordenada, una tras otra.
