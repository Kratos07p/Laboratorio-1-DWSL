Introducción a PHP

PHP (Hypertext Preprocessor) es un lenguaje de propósito general enfocado al desarrollo del lado del servidor. Su fortaleza radica en generar contenido dinámico, procesar formularios y comunicarse con bases de datos para construir aplicaciones web completas.

Las estructuras de control te permiten decidir qué instrucciones ejecutar y cuántas veces repetirlas. En la práctica, combinarlas correctamente es lo que transforma páginas estáticas en sistemas que reaccionan a datos y eventos del usuario.

A continuación se detallan las estructuras que ya has visto en esta página, con una explicación más profunda de su propósito, cuándo usarlas y buenas prácticas.

IfCondicional

La sentencia if sirven para ejecutar un bloque de código de forma condicional, es decir, solo si se cumple una determinada expresión o condición que evalúa a verdadero (true), y esto permite al programa tomar decisiones, definir comportamientos distintos para diferentes escenarios y controlar el flujo de ejecución de la aplicación web.

Cuándo usarla: validación de entradas (edad mínima, campos requeridos), control de permisos, bifurcar lógica según estado (p.ej., autenticado/no autenticado).

Buenas prácticas: mantén las condiciones legibles (usa variables con nombres claros), evita anidar muchos if seguidos si una estructura switch o una tabla de decisiones lo simplifica, y extrae lógica compleja a funciones. 

<?php
$numero = 10;
if ($numero > 5) {
  echo "El número es mayor que 5";
}
?>
SwitchSelección múltiple

switch compara el valor de una misma variable contra múltiples opciones predefinidas (cases). Es ideal cuando varias rutas dependen de un único dato (p. ej., tipo de usuario, opción de menú). Cada case termina usualmente con break para evitar que la ejecución “caiga” al siguiente caso.

Cuándo usarlo: mapas de opciones claras (1, 2, 3...), constantes simbólicas, normalizar acciones según un estado.

Buenas prácticas: agrupa case que compartan lógica; usa default para manejar entradas no contempladas; procura que cada caso sea corto y delegue en funciones cuando haga falta. 

<?php
$dia = "lunes";
switch ($dia) {
  case "lunes":
    echo "Primer dia de semana";
    break;
  case "martes":
    echo "Segundo dia de semana";
    break;
  default:
    echo "Día no válido";
}
?>
WhileBucle

while repite un bloque mientras una condición sea verdadera. La verificación ocurre antes de ejecutar el cuerpo; si la condición es falsa al inicio, el bucle no corre ninguna vez. Es útil cuando no conoces de antemano cuántas iteraciones harás y dependes de una condición que puede cambiar dentro del ciclo.

Cuándo usarlo: lectura de datos hasta fin de archivo, espera de un estado, intentos controlados con contador.

Buenas prácticas: garantiza que la condición cambie dentro del bucle para evitar bucles infinitos; limita el número de iteraciones cuando haya riesgos (timeouts, recursos). 

<?php
$i = 1;
while ($i <= 100) {
  echo $i;
  $i++;
}
?>
Variante: do...while

do…while ejecuta primero el bloque y luego evalúa la condición; por eso se ejecuta al menos una vez. Útil para menús interactivos o procesos que deben ocurrir una vez y repetirse si el usuario lo decide. 

<?php
$j = 1;
do {
  echo $j . " ";
  $j++;
} while ($j <= 5);
?>
ForBucle

for se usa cuando conoces (o puedes calcular) el número de repeticiones. Integra en una sola línea la inicialización del contador, la condición de continuación y la actualización del contador, lo que lo hace compacto y fácil de leer para secuencias numéricas.

Cuándo usarlo: recorrer rangos numéricos, generar series, ejecutar un bloque un número concreto de veces.

Buenas prácticas: evita lógica pesada en la condición y en la actualización; si el cuerpo se vuelve complejo, extrae funciones para mantener claridad. 

<?php
for ($i = 1; $i <= 10; $i++) {
  echo $i;
}
?>
Control de flujo

Dentro de bucles puedes alterar el curso con continue (salta a la siguiente iteración) y break (termina el bucle). Úsalos con criterio para no ocultar la intención del código. 

<?php
for ($i = 1; $i <= 10; $i++) {
  if ($i === 3) continue; // salta el 3
  if ($i === 8) break;    // corta en 7
  echo $i . " ";
}
?>
ForeachColecciones

foreach recorre de forma simple los elementos de un arreglo (y estructuras iterables). Es la opción más clara cuando te interesa cada elemento sin manejar índices manualmente. Permite obtener solo el valor o clave y valor en arreglos asociativos.

Cuándo usarlo: procesar listas de resultados, renderizar elementos en vistas, transformar colecciones.

Buenas prácticas: al modificar elementos usa referencias (&$var) y recuerda unset() para liberar la referencia; evita mutar la colección mientras la recorres salvo que sepas exactamente el efecto. 

<?php
$colores = ["rojo", "verde", "azul"];
foreach ($colores as $color) {
  echo $color;
}
?>
